Match Detail

A match page should tell users what is happening immediately

After a user arrives from the schedule or live page, the first need is obvious: scoreline, key events, and the shape of the match.

The match page is the strongest candidate for real-time API data, and the best place to flow users into teams, prediction, and deeper content.
