Prediction Hub

A prediction page should explain the model, not just throw out a number

This page should explain how the model sees football before showing the probabilities. The result matters, but so does knowing why it exists and where not to over-trust it.

Run a matchup Review historic upsets

A useful prediction page is not about “the model has spoken.” It is about how much confidence a user should assign to what they just read.

Model position This is an explainable Elo model, not betting market odds and not a black-box AI.
Best use case It is strong for previews, team comparisons, and editorial discussion, not live betting decisions.
Main value It turns historical data into a real tool and interaction layer instead of leaving it as a static archive.
